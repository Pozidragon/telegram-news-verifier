# Telegram News Verifier

Master's thesis project for automated formation of a verified Ukrainian language corpus based on Telegram messages and trusted news sources.

## Features
- Telegram data collection
- News collection from RSS feeds
- Text preprocessing
- Verification using TF-IDF + cosine similarity
- Verification using sentence-transformers
- Experimental evaluation with manual labeling

## Tech stack
- Python
- Telethongit init
- BeautifulSoup
- scikit-learn
- sentence-transformers
